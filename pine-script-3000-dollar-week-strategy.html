Strategy Overview

This Pine Script strategy targets breakouts on the Russell 2000 index, capturing momentum surges with minimal lag using the following logic:

  1. 20-period High/Low breakout confirmation
  2. Volume spike filter to avoid fakeouts
  3. ATR-based trailing stop-loss to ride trends

Complete Pine Script Code


//@version=5
strategy("Breakout Momentum RUT Strategy", overlay=true, 
         initial_capital=25000, default_qty_type=strategy.percent_of_equity, 
         default_qty_value=15, commission_type=strategy.commission.percent, 
         commission_value=0.1)

// Breakout Conditions
highestHigh = ta.highest(high, 20)
lowestLow = ta.lowest(low, 20)
volumeAvg = ta.sma(volume, 20)

long_condition = close > highestHigh[1] and volume > volumeAvg
short_condition = close < lowestLow[1] and volume > volumeAvg

// ATR-based trailing stop
atr = ta.atr(14)
trail_offset = 2.0 * atr

// Execute strategy
if long_condition
    strategy.entry("Breakout Long", strategy.long)
    strategy.exit("Exit Long", "Breakout Long", trail_points=trail_offset)

if short_condition
    strategy.entry("Breakout Short", strategy.short)
    strategy.exit("Exit Short", "Breakout Short", trail_points=trail_offset)
