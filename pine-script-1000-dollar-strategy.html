Indicator Combination

The strategy uses a unique combination of:

  1. Volume-Weighted Moving Average (VWMA): 20-period for trend direction
  2. Modified RSI: 3-period with 30/70 thresholds
  3. Order Flow Imbalance: Detects institutional buying/selling

Complete Pine Script Code

//@version=5
strategy("$1000/Day Strategy", overlay=true, margin_long=100, margin_short=100)

// Input parameters
vwmaLength = input(20, "VWMA Length")
rsiLength = input(3, "RSI Length")
entryThreshold = input(30, "Entry Threshold")
exitThreshold = input(70, "Exit Threshold")

// Calculate indicators
vwma = ta.vwma(close, vwmaLength)
rsi = ta.rsi(close, rsiLength)
volumeImbalance = (close - open) / (high - low) * volume

// Entry conditions
longCondition = close > vwma and rsi < entryThreshold and volumeImbalance > 1.5
shortCondition = close < vwma and rsi > exitThreshold and volumeImbalance < -1.5

// Execute trades
if (longCondition)
    strategy.entry("Long", strategy.long)
if (shortCondition)
    strategy.entry("Short", strategy.short)

// Exit conditions
strategy.close("Long", when = rsi >= exitThreshold)
strategy.close("Short", when = rsi <= entryThreshold)
